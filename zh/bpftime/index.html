Product / Runtime

bpftime

高性能 userspace eBPF runtime 和 extension framework，面向 production extension、observability 和 GPU-aware instrumentation。

阅读文档GitHub企业支持

Use cases

  • Low-overhead tracing
  • Custom runtime extension
  • uprobe / syscall / USDT / XDP / GPU paths
  • Production integration

Commercial support

  • Enterprise support
  • Integration with existing observability or runtime platforms
  • Performance tuning and benchmarking
  • Custom runtime / runtime extension engineering

商业支持范围

围绕 bpftime 开源 runtime 提供生产集成、性能调优和定制 runtime 工程支持。

Enterprise support

生产集成和维护

帮助团队把 bpftime 集成到已有 tracing、networking、sandbox 或 runtime extension 工作流中。

Performance

低开销观测和调优

围绕 uprobe、syscall、USDT、XDP 或 GPU 相关路径做 benchmark、profiling 和性能优化。

Custom runtime

定制事件源和扩展

为特定系统构建 attach path、helper、map、policy 或部署模型，而不强迫用户改动业务代码。
