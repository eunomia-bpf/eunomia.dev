如何保持索引更新

仓库每周运行一次定时审计，把本页与近期 arXiv 记录比较，同时检查论文和 artifact 链接、中英文覆盖以及 Blog 状态。发现 drift 后，自动任务会创建或更新同一个 GitHub issue。Maintainer 在发布论文相关变更前，也可以手动运行同一项审计。
