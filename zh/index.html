Eunomia 提供什么

Eunomia 的工作集中在 runtime infrastructure、公开资源和 AI agent systems。

Runtime extension

让 eBPF 不只停留在内核工作流里

通过 userspace execution、GPU paths、JIT/AOT 支持和部署模型，把 eBPF 扩展到生产工具链可以直接使用的系统路径。

Resources

发布可复用的教程、文档、博客和论文

面向 eBPF 开发者维护可执行教程、文档入口、技术文章和研究产物。

AI Agents

让 agent 行为可观测、可控制

在系统边界监控 agent，关联进程和网络行为，并保护依赖 checkpoint/restore 的工作流。
