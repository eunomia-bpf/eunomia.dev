# Semantic Flamegraph Research Note

This directory keeps one public, paper-shaped research note for semantic agent
flamegraphs and `agentpprof`. It intentionally does not include the large
generated experiment-output tree from the research branch or the many interim
planning notes used while developing the idea.

Read:

- [paper/evaluation-claims-setup.zh-CN.md](paper/evaluation-claims-setup.zh-CN.md):
  Chinese write-up of the claims, terminology, experiment setup, oracles,
  results, and evidence boundaries.

Generated files under `docs/visexp/out/` are not committed to `main` here. They
remain on the research branch as experiment artifacts.
